How to Run it in Your Tesla

1 Save in xLights

Open `lightshow.xsq` inside the **xLights** software. Verify that `lightshow.wav` is loaded as media and save it.

2 Export Sequence

Export the sequence in xLights to get your **`lightshow.fseq`** file (which contains the binary commands for the lights).

3 Plug & Play

Copy `lightshow.fseq` and `lightshow.wav` (renamed to `lightshow.mp3` if needed) into a folder named **`LightShow`** at the root of a USB flash drive, plug it into your car, and start the Show!
